# Starcraft II Improved Ladder - Technical Road Map

Starcraft II Improved Ladder is a matchmaking service for the game Starcraft II.

## v1.0.0 Feature list

- Features:
  - Allow a user to login to the site.
  - Allow a user to create and find unranked custom games.
  - Allow a user to find ranked ladder matches.
  - Allow a user to upload match results.
- Pages:
  - Lobby: Find unranked custom games.
  - Ladder: Find ranked ladder games.
  - Leaderboard: View status of ladder.
  - Stats: Data analysis of the ladder.
  - FAQ: How-to and reasoning.
  - Login: Allow a user to login via OAuth/OIDC.
  - Register: Allow a user to register. Not sure this is needed if using OAuth/OIDC.
  - Settings: Allow a user to set preferences and edit account details.
  - Not Found: Handle routing exceptions.
